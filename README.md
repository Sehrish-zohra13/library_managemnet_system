# Lib-Link – Digital Library Management System

## Overview

Lib-Link is a full-stack library management system developed during a 24-hour hackathon. It is designed to simplify book tracking, issuing, and user management using modern web technologies.

---

## Features

* Book Catalog and Search
* QR Code Scanning for Book Identification
* Student Dashboard
* Admin Control Panel
* Reservation System
* Fine Management
* Authentication System

---

## Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express.js
* Database: MySQL
* Tools: Git, GitHub

---

## Project Structure

* `/shelfsense` → Backend server and APIs
* `/stitch` → UI designs and screens
* `database_schema_mysql.html` → Database schema

---

## Setup Instructions

